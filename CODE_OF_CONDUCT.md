# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its Code of Conduct.

Full text and details:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## In short

- Be kind. Be patient.
- Disagree on technical merit; never on personhood.
- English is the primary language for project communication; other languages welcome in informal discussion.

## Reporting issues

Concerns about community interactions can be reported privately to the maintainers via GitHub Security Advisory or the email listed in `SECURITY.md`. We commit to confidential handling and a response within 7 days.
